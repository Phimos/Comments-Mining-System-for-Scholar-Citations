# Comments-Mining-System-for-Scholar-Citations

Yunchong Gan's Final Design

## Installation

```
pip install -e .
```

## Usage

**collect metadata and pdf files**
```
python collect_pdf.py
```

**fill info and generate summary**
```
python generate_summary.py
```
